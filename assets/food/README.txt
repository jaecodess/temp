Food Images for Nutrition Quest
==============================

Place PNG images here to display actual food pictures instead of circles.

Good food (name the file exactly as shown):
  apple.png, banana.png, broccoli.png, carrot.png, orange.png, grapes.png, spinach.png, tomato.png

Bad food (optional):
  burger.png, soda.png, chips.png, candy.png, pizza.png

Images should be square or nearly square. Recommended size: 64x64 or 128x128 pixels.

If an image is missing, the game will show a colored circle with the food name instead.
